# Documentação do Projeto

Use esta divisão como regra prática de navegação:

- `docs/guias/`: material operacional e guias de uso
- `docs/decisoes/`: decisões estáveis de arquitetura e produto
- `docs/learnings/`: descobertas, auditorias e aprendizados acumulados
- `docs/planos/`: propostas e planos de execução ainda em aberto
- `docs/reports/`: relatórios versionados que valem como artefato de análise

Arquivos históricos de validação com imagens e rodadas continuam nas pastas temáticas já existentes em `docs/`.
