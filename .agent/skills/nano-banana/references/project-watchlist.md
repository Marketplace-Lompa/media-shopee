## Datasets and benchmarks

- Valuable when the user asks about evaluation, editing benchmarks, or multi-turn editing research.

- Useful for examples, wrappers, and quick experimentation patterns.

- `https://github.com/ConechoAI/Nano-Banana-MCP`

## Curated prompt and ecosystem collections

### awesome-nano-banana

## Interpretation guidance

When citing community projects:
- state whether the project is official, community, or dataset-oriented
- avoid assuming the repo reflects the latest Google API surface
- use official Google docs to verify every model ID, parameter, or limit
- treat prompt collections as examples, not specifications
